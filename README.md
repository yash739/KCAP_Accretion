## Abstract

Accretion is a fundamental process by which gravitational potential energy gets reprocessed into light of various wavelengths, with kinetic energy as the go-between. Yet in this process angular momentum is conserved, so how exactly do bodies lose energy in this process? This project explores the fundamental physics of accretion, especially disks, and makes some forays into simulations of accretion disks.
